# 卡片排版规范

## 源代码格式

### 所有多行字符串用数组 + join

```ts
// ✗ 禁止
body: '第一行\n\n第二行\n第三行',

// ✓ 正确
body: [
  '第一行',
  '',
  '第二行',
  '第三行',
].join('\n'),
```

适用字段：`body`、`code`、`question`。原则：源码里一行 = 渲染后一行。

### 代码字符串

每行 C++ 代码独立一个数组元素：

```ts
code: [
  '#include <iostream>',
  '',
  'int main() {',
  '  std::cout << "Hello";',
  '  return 0;',
  '}',
].join('\n'),
```

`\\n`（C++ 字符串里的换行）保留在对应行内，不要拆。

### 练习题

题目文字和代码分开：

```ts
question: [
  '以下代码输出什么？',
  '',
  'for (int i = 0; i < 3; i++) {',
  '    cout << i << " ";',
  '}',
].join('\n'),
```

## 排版规则

### 禁止左右并列

手机上列宽不够，一律上下排列。包括：
- 对比代码（你写的 vs 编译器翻译的）
- 真值表
- 运算符表格

```ts
// ✗ 左右并列
'你写的：              编译器翻译成：\n' +
'for (int v : arr)     int n = ...',

// ✓ 上下排列
'你写的：',
'  for (int v : arr) {',
'      cout << v;',
'  }',
'',
'    ↓ 编译器自动翻译成 ↓',
'',
'  int n = 元素个数;',
'  for (int i = 0; i < n; i++) {',
'      int v = arr[i];',
'      cout << v;',
'  }',
```

### 真值表竖排

```ts
// ✗ 横排
'1 && 1 = 1    1 && 0 = 0    0 && 1 = 0    0 && 0 = 0',

// ✓ 竖排
'A && B 真值表：',
'  1 && 1 = 1',
'  1 && 0 = 0',
'  0 && 1 = 0',
'  0 && 0 = 0',
```

### 行内注释

代码行的 `//` 注释不要挤在同一行末尾，尤其行很长时。挪到代码块下面单独写。

```ts
// ✓ 注释单独成行
'for (int i = 1; i <= 3; i++) {',
'    for (int j = 1; j <= 3; j++) {',
'        if (j == 2) break;',
'        cout << i << "," << j << "  ";',
'    }',
'}',
'// 输出：1,1  2,1  3,1',
```

## 字体

代码字号 12px，通过 `FontSize.code` token 引用：

```ts
import { FontSize } from '@/theme';
// ...
fontSize: FontSize.code,
lineHeight: 19,
```

## 检查清单

写完一个节点后：

- [ ] 有没有 `\n` 堆在一行的字符串？（源码可读性）
- [ ] 有没有左右并列排版？（手机适配）
- [ ] 练习题和前面例子有没有重复？（内容不冗余）
- [ ] 代码行宽会不会超出手机屏幕？（字号12、避免长行）
